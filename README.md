# Soccer Forecasting Tool

A lightweight Python + Streamlit scaffold for building a soccer match forecasting app.

## Project Structure

- `README.md` - project overview and usage
- `requirements.txt` - Python dependencies
- `app.py` - Streamlit app stub

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run the App

```bash
streamlit run app.py
```
